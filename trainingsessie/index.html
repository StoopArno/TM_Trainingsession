Transitions

CSS

.transition{
transition: width 2s ;
}
.transition:hover{
width: 500px;
}

CSS

.transition{
transition: width 2s, height 5s;
}
.transition:hover{
width: 500px;
height: 300px;
}

Properties

transition-timing-function: (linear, ease, ease-in, ease-out, ease-in-out);
transition-delay: Xs;
transition-property: width;
transition-duration: Xs;

2D Transformations

}
.translate:hover{

}
