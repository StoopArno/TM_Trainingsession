Code

 animation: vb1 4s infinite;
}

@keyframes vb1{
 0%{transform: translateX(0);}
 50%{transform: translateX(300px);}
 100%{transform: translateX(0);}
}

Oefening animation

In deze oefening moet je zorgen dat het blokje 600px naar rechts en 100px naar onder verschuift. Vervolgens gaat het weer 100px naar boven. Daarna gaat het terug 600 naar links en 100 naar onder om daarna terug uit te komen op de oorspronkelijke toestand.
Bij elke verplaatsing draait het blokje telkens 45°.

  • Oplossing

    .oef-animation-1{
     animation-name: oef1 3s infinite;
     animation-duration: 3s;
     animation-iteration-count: infinite;
    }

    @keyframes oef1{
     25%{transform: translate(600px, 100px) rotate(45deg);}
     50%{transform: translate(600px, 0px) rotate(90deg);}
     75%{transform: translate(0px, 100px) rotate(135deg);}
     100%{transform: translate(0, 0) rotate(180deg);}
    }

Dit is een auto met draaiende wielen die van links naar rechts rijdt

Uitbreiding: Wheelie

2e uitbreiding: De auto start in normale positie en eindigt in een wheelie.
